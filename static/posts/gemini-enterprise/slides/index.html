Use Case · Salesforce
영업 현장의 세일즈포스 에이전트
Problem

영업 담당자는 CRM 조회·리포트 추출·파이프라인 요약에 하루 수시간을 소모합니다.

Solution

세일즈포스 커스텀 에이전트가 자연어 질문을 SOQL로 변환해 로그인 사용자·영업 기회·계약을 즉시 조회·요약합니다.

Impact
  • ✓ 파이프라인 조회 시간 90% 단축
  • ✓ 담당자가 즉시 의사결정에 필요한 정보 확보
  • ✓ Multi-Currency(KRW) 정규화 자동 처리

Prompting Best Practices · 모든 사용자 대상
좋은 프롬프트의 4가지 조건
✗Weak Prompt
"프로젝트 계획 만들어줘"
· 역할(Persona) 지정 없음 · 참조할 맥락(Context) 누락 · 산출물 형식(Specificity) 불명 · 범위·기간이 모호
→
✓Powerful Prompt
Persona당신은 시니어 프로젝트 매니저입니다. Context첨부 문서의 맥락을 사용하세요. Task이번 분기 신규 앱 런칭을 위한 프로젝트 계획을 만들어 주세요. SpecificityTask · Owner · Due Date 컬럼을 가진 표 형식으로 출력하세요.
✓ 역할 · 맥락 · 과제 · 형식이 모두 명확 · 출력 포맷이 일관되고 후속 활용 용이
